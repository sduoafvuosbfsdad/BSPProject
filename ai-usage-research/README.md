# AI工具使用研究 - 中国与新加坡公民AI能力研究

## 研究概述

本研究全面分析中国和新加坡公民在AI工具（包括大型语言模型LLMs）使用方面的情况，重点关注**可及性**、**使用人口规模**和**熟练程度**三个维度。

---

## 研究背景

| 维度 | 描述 |
|------|------|
| **时间范围** | 2023-2026年 |
| **目标国家** | 中国、新加坡 |
| **研究主题** | 公民日常生活与工作中AI/LLM使用情况 |
| **输出语言** | 中文 |
| **引用格式** | APA格式 |

## 研究方法

1. **数据收集**：官方政府报告、国际研究机构数据、学术研究
2. **交叉验证**：使用多个来源验证关键统计数据
3. **时效性检查**：确保所有数据在2023-2026年范围内
4. **来源可信度**：优先使用官方政府报告和国际权威机构数据

---

## 主要信息来源

### 官方政府来源
- **中国**：CNNIC（中国互联网络信息中心）、中国信通院、中央网信办、教育部
- **新加坡**：IMDA（资讯通信媒体发展局）、Smart Nation数字政府办公室、教育部

### 国际研究机构
- KPMG（全球AI信任调查）
- IDC（大模型应用市场份额报告）
- Access Partnership（消费者AI调查）
- Ipsos（民意调查）
